# Vishal Logistics

Marketing site for Vishal Logistics — New Delhi freight forwarders. Ocean & air freight,
customs clearance, and door-to-door delivery.

## Stack

- [Vite](https://vitejs.dev) + React 19
- [Tailwind CSS v4](https://tailwindcss.com)
- [motion](https://motion.dev) for animation, [lucide-react](https://lucide.dev) for icons
- react-router v7 (routes: `/`, `/about` services, `/contact`)

## Develop

```sh
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # serve the production build
```

Contact details, office addresses, and the service list live in `src/site.js`.
